# Anorbank Card Comparison App

Интерактивное веб-приложение для сравнения условий и характеристик банковских карт ANORBANK. Позволяет пользователям наглядно сопоставлять тарифы, лимиты на снятие наличных, кэшбэк и стоимость SMS-информирования для выбора оптимального финансового продукта.

## 🚀 Функционал проекта

* **Декомпозированная архитектура:** Интерфейс разделен на изолированные, переиспользуемые компоненты (`Header`, `CardGrid`, `ComparisonTable` и др.).
* **Централизованный стейт:** Логика выбора карт вынесена в глобальное хранилище **Zustand**.
* **Persistent State (Сохранение данных):** Выбранные карты автоматически сохраняются в `localStorage` с помощью middleware `persist`. Данные не сбрасываются при перезагрузке страницы ($F5$).
* **Умная таблица сравнения:** * Динамическая подсветка строк, в которых значения характеристик различаются между выбранными картами.
  * Ограничение на выбор (не более 3 карт одновременно) с валидацией внутри стора.
  * Защита рантайма от падения (`RangeError`) при заполнении пустых слотов таблицы.
* **Полный адаптив:** Удобная сетка на десктопе и трансформация таблицы в стек карточек на мобильных устройствах (< 640px).
* **Строгая типизация:** Полное покрытие интерфейсов на TypeScript с учетом конфигурации `"verbatimModuleSyntax": true`.

## 🛠 Технологический стек

* **Frontend:** React 18, TypeScript
* **State Management:** Zustand + Persist middleware
* **Styling:** Tailwind CSS
* **Build Tool:** Vite
